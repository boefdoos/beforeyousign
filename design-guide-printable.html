Design System

Serene & Minimal

This guide documents the visual language and component patterns for beforeyousign.be. Follow these principles to maintain consistency across all tools in the platform.

Version 1.0
November 2025

Core Principles

The Feeling

Think: A well-made book on your desk. Quiet, confident, always available when you need it.

NOT: Flashy SaaS dashboard, aggressive call-to-action, gamified experience
YES: Calm tool, editorial quality, timeless aesthetic, professional but warm

Three Rules

  • Subtract, don't add — Remove before you embellish
  • Whitespace is content — Generous margins create calm
  • Serif = serious — Typography carries the mood

Usage Guidelines

  • ✓ Use paper (#faf9f7) for all page backgrounds
  • ✓ Use ink (#2d3436) for primary text content
  • ✓ Use accent (#6c7a89) for interactive elements
  • ✗ Never use pure black (#000000)
  • ✗ Never use pure white (#ffffff) as background
  • ✗ Avoid bright, saturated colors

Typography

Font Stack

font-family: 'Georgia', 'Times New Roman', serif;

Why serif? Creates calm, editorial feeling. Signals: "This is considered, not rushed."

Font Sizes

Use sparingly — these are the ONLY sizes you need

text-xs 9px Footnotes, legal text
text-sm 10px Labels, captions
text-base 11px Body text
text-lg 14px Subheadings
text-xl 18px Section titles
text-2xl 24px Page titles
text-3xl 32px Hero elements
text-4xl 48px Large scores/numbers

Font Weight & Style

Normal (400) — Default for all text
Bold (700) — Use sparingly for emphasis
Italic — Use for metadata, quotes, subtle info

Line Height

line-height: 1.6; /* DEFAULT - use everywhere */ line-height: 1.4; /* Only for large text (>32px) */

Spacing System

Base Unit: 4px

All spacing must be a multiple of 4px. This creates visual rhythm and consistency.

p-1 4px
p-2 8px Within components
p-3 12px Related items
p-4 16px Between items
p-6 24px Small sections
p-8 32px Medium sections
p-12 48px Large sections
p-16 64px Major sections

Interactive Elements

Primary Button

<button className=" w-full py-4 px-6 bg-[#6c7a89] hover:bg-[#5a6877] text-white font-serif text-sm rounded-sm transition-colors "> Button Text </button>

Secondary Button

<button className=" px-6 py-3 text-[#636e72] hover:text-[#2d3436] font-serif text-sm border border-[#e8e6e1] hover:border-[#6c7a89] rounded-sm transition-colors "> Button Text </button>

Input Field

<input className=" w-full px-4 py-3 bg-white border-2 border-[#e8e6e1] focus:border-[#6c7a89] rounded-sm font-serif text-base transition-colors " />

Interaction Guidelines

  • ✓ Use rounded-sm (2px corners only)
  • ✓ Hover states darken 10-15%
  • ✓ Transitions ≤ 200ms
  • ✗ No drop shadows on buttons
  • ✗ No bounce, pulse, or wiggle effects

Layout Patterns

Max Width: 672px

Almost always use max-w-2xl (672px) for content width. This is the optimal reading width.

<div className="max-w-2xl mx-auto"> {content} </div>

Page Structure

<main className="min-h-screen bg-[#faf9f7] py-20 px-6"> <div className="max-w-2xl mx-auto"> <header className="mb-16"> {/* Title + subtitle */} </header> <div className="space-y-12"> {/* Content sections */} </div> </div> </main>

Component Patterns

Content Box

<div className=" p-6 bg-white border border-[#e8e6e1] rounded-sm "> {content} </div>

Warning Box

<div className=" p-6 rounded-sm bg-[#fdf9f5] border-l-2 border-l-[#d4a574] "> <h3 className="font-serif font-bold mb-2"> Warning Title </h3> <p className="text-sm font-serif"> Description </p> </div>

List with Checkmarks

<div className="space-y-3"> {items.map(item => ( <div className="flex gap-3"> <span className="text-[#7fa99b]">✓</span> <p className="text-sm font-serif"> {item} </p> </div> ))} </div>

Branding

  • ✓ beforeyousign.be

Taglines

Pick one per context:

  • • "voor je tekent" (NL)

Tone of Voice

  • Descriptive, not prescriptive — "This clause is unusual" not "You should reject this"
  • Calm, not urgent — "Consider reviewing" not "WARNING! ACT NOW!"
  • Educational, not sales — "Here's what this means" not "Sign up for premium"

Pre-flight Checklist

Before shipping any new feature, verify:

Colors

  • □ Only uses palette colors (no random hex codes)
  • □ Paper background (#faf9f7) on main container
  • □ Text is ink (#2d3436) or ink-light (#636e72)
  • □ No pure black or pure white

Typography

  • □ Uses serif font (Georgia/Times)
  • □ Font sizes from the scale only
  • □ Line-height is 1.6 for body text
  • □ Headers use tracking-tight or tracking-wider

Spacing

  • □ All spacing in 4px increments
  • □ Generous whitespace between sections (48-64px)
  • □ Consistent vertical rhythm
  • □ Max-width: 672px (max-w-2xl) for content

Interactions

  • □ Hover states are subtle (10-15% darker)
  • □ Transitions are 200ms or less
  • □ No bounce/pulse animations
  • □ Buttons use rounded-sm (not rounded-full)

Branding

  • □ Uses "beforeyousign.be" (lowercase)
  • □ Tone is calm and educational
  • □ No aggressive CTAs or urgency

Remember: Every pixel should earn its place.
When you add something, ask "Does this make it more calm?"
If not, don't add it.
